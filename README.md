# Hello World Program

This is a simple "Hello World" program written in Python.

## Files

- `hello_world.py` - The main Python script that prints "Hello, World!" to the console

## How to Run

To run the program, open a terminal in this directory and execute:

```bash
python hello_world.py
```

## Requirements

- Python 3.x installed on your system

## About

This is a basic program that demonstrates:
- Python script structure
- Function definitions
- Print statements
- Main function pattern

Perfect for getting started with programming!
